# Rosary with Sacred Art - Media Assets

This repository hosts media files that were modified from the originals by Kevin Gimpel for use in the [Rosary with Sacred Art](https://github.com/kgimpel/rosary-with-sacred-art) app. They are released under the same licenses they were released under originally.

Modifications include volume normalization, trimming, and re-encoding.
